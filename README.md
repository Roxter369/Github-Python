# Información sobre Python
Python es un lenguaje de programación el cual puede ayudar a crear:
* Programas 
* Códigos
* Videojuegos
* Calculadoras
* Entre otros.
Las ventajas de utilizar Python son:
* Fácil de entender
* Intuitivo
* Rápido de programar
* Es un lenguaje de alto nivel
* Multiplataforma
